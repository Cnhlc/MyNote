# MyNote
1.	功能要求
（1）	支持新增、修改和删除日记。日记为纯文本格式，数据库表应该至少包含日记标题，日记内容，日记的创建时间，日记的作者等信息。
（2）	显示当前用户的日记列表，点击某一条记录，可以打开一个新的界面供用户查看、编辑日记。
（3）	设计用户界面，维护日记的作者信息（至少包含作者的姓名）。
（4）	允许用户添加照片到日记中，方法是允许用户通过拍照或读取系统相册的方式添加照片，并将选中的照片拷贝到日记本应用程序的内部存储目录下，将照片文件名保存入数据库中与对应的日记记录关联在一起。用户查看日记时也显示日记中包含的照片。
2.	非功能要求
（1）	保证日记的安全性，设置用户登录密码，及确认密码操作。
（2）	软件要可靠，不能出现闪退或者无法点击操作的情况。
3.	设计约束
（1）	以ListView的形式显示当前用户的日记列表。
（2）	维护日记的作者信息（至少包含作者的姓名），并存入SharedPreferences中。
（3）	日记的内容存储在SQLite数据库中。
